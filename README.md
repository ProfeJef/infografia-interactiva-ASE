# 🏫 Currículo Vivo: Infografía Interactiva (RPG Style)

Este proyecto es una infografía pedagógica gamificada con estética de videojuego RPG 2D. Su objetivo es visualizar y explorar la propuesta sobre el **uso del tiempo libre en escuelas primarias**, integrando el Aprendizaje Social y Emocional (ASE) en el descanso formativo.

## 📂 Estructura del Proyecto

/mi-proyecto-infografia
│
├── index.html                 # Estructura base y contenedores globales
├── README.md                  # Documentación técnica
│
├── /css
│   ├── main.css               # Estilos globales, tipografías y layout base
│   └── rpg-ui.css             # UI RPG: barras de estado, modales, animaciones
│
└── /js
    ├── app.js                 # Punto de entrada (ES6 Module)
    ├── data.js                # Datos pedagógicos de las estaciones
    └── game-logic.js          # Lógica de juego, movimiento y UI

## 🚀 Cómo desplegar en GitHub Pages

1. Crea un nuevo repositorio en GitHub.
2. Clona el repositorio en tu máquina local: `git clone <tu-url-repo>`.
3. Copia todos los archivos de esta estructura directamente en la raíz del repositorio.
4. Haz commit y push de los cambios (`git add .`, `git commit -m "Initial commit"`, `git push`).
5. En GitHub, ve a **Settings > Pages**.
6. En **Source**, selecciona la rama `main` y la carpeta `/root`, luego guarda.
7. ¡Listo! En unos minutos tu infografía interactiva estará viva en la URL proporcionada por GitHub.
